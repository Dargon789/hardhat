# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

We take security vulnerabilities seriously. Please report any security vulnerability by [reporting it privately on GitHub](https://github.com/NomicFoundation/hardhat/security/advisories/new).

You can expect a response from us within 48 hours. If the issue is confirmed, we will work on a fix and coordinate with you on the public disclosure.
